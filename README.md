## On Linux server

1. install make command  
   `sudo apt update && sudo apt install make -y`

2. build dockerfild  
   `make build`

3. Start DB & PHPMyadmin  
   `make up-db`

4. Start GO app  
   `make up-go`

5. Checking docker  
   `docker-compose ps`

6. Log go app  
   `docker-compose logs go-ex`
6. Log go app real time  
   `docker-compose logs -f go-ex`
